# 關於本專案

本專案是碳排放計算機，可依據所選國家與行為類別的不同，計算其每年平均每人碳排放。

# 如何使用本專案

1. 畫面上方共有六個排碳類別按鈕：平均碳足跡、飲食排碳、航行排碳、駕車排碳、穿著排碳、能源排碳。請選擇欲查詢之排碳類別。
2. 選擇排碳類別後，請再選擇欲查詢之國家。
3. 選擇國家後，若該排碳類別仍有需要選擇或輸入之資料，請完成選擇或輸入程序。
4. 最後可得到排碳量。
